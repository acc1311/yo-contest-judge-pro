# Utilizare rapida — YO Contest Judge PRO v2.2

## Flux de lucru tipic

1. **Import loguri** (Ctrl+O) — ADIF, Cabrillo, CSV, JSON, EDI
2. **Alege concurs** din dropdown
3. **Validare** (F5) — RST, data, banda, mod, judet, locator, serial
4. **Cross-Check** — NIL, Busted Call/Band/Time
5. **Arbitraj complet** — scor final cu penalizari
6. **Export** — HTML / CSV / TXT / JSON / **PDF**
